##Abstract

本研究(TERM)の最終ゴールは、「カメラトラップデータ群に対して、分類器が自信満々が自信満々に誤判定してしまう危険を減らし、判定を棄却すべき場面では安全にOpen-Setし、さらにその原因を現場対処可能な形で返すこと」である。

本特プロは、「動物種」の分類と「原因」の2モデルを分けて作成し統合する手法を提案した上で、それを構成する一部(環境/原因要因の分類)をQwen3-VLモデルで構築した。

評価は個人の主観であるが、概ね分類することができた。これを発展させ知識蒸留条件下で、Unknown棄却、再確認フロー、行動提案を一体化させることをTERMまでの目標とする。

{仮説1}
本研究では，野生動物画像の誤判定の主因が，分類性能不足そのものだけではなく，夜間・遮蔽・低画質・セマンティックな曖昧さなどを含む「本来は無理に分類すべきでない画像」を既知クラスへ押し込んでしまう点にあると考える．
この仮説に基づけば，動物分類器の高精度化だけでは不十分であり，撮影条件や認識困難性を診断する機構を統合することが重要となる．

以上より，本研究の問題設定は，カメラトラップ画像に対して，分類器が自信をもって誤判定してしまう危険を低減し，棄却すべき場面では安全にUnknownとし，さらにその原因を利用者が理解し対処できる形で提示することである．
